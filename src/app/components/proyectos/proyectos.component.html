Pagina Compras

Este proyecto de pagina es parte del curso de Dalto en el cual utilizando JavaScript realize una pagina en la cual se introducia un producto (es este caso solo llaves), al cual se le genera una id automaticamente y manualmente desde una base de datos se añade el precio y nombre del producto.

Pagina Starbucks

En este proyecto realize una pagina la cual tiene un login y la posibilidad de crear una cuenta, tambien cuenta con noticias o articulos los cuales se pueden actualizar facilmente, ademas tiene menus varios donde se pueden ver los productos de tal empresa y conseguir codigos para usar en la misma.

Portfolio de Prueba
